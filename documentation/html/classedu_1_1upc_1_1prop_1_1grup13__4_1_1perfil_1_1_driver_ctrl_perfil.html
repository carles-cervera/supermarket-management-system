Detailed Description

Classe Driver per a gestionar i provar les operacions de perfils d'usuari.

Aquesta classe proporciona una interfície de text per a provar les operacions relacionades amb els perfils, com ara crear un perfil, iniciar sessió, canviar contrasenya, recordar contrasenya, tancar sessió i eliminar perfils.
