Detailed Description

Panell per crear una distribució

Aquest panell proporciona una interfície gràfica per introduir l'ID de la distribució, seleccionar un mètode (Aproximat o Backtracking) i especificar el nombre de prestatges. Inclou opcions per crear la Distribució o tancar aquesta finestra.
