Detailed Description

La classe ScriptDialogViewController és un SINGLETON i gestiona la interacció entre la finestra ScriptDialog i la lògica de presentació. S'encarrega d'afegir els esdeveniments als components de la finestra, importar el script introduït a través del textArea, i tancar la finestra quan sigui necessari.

◆ importar()

void edu.upc.prop.grup13_4.presentacio.ScriptDialogViewController.importar ( )

Importa el script introduït a través del textArea i l'analitza.

Precondition
El text del script ha de ser introduït a la finestra de text (textArea).
Postcondition
Si el text no és buit, es crea un parser per analitzar el script i es criden les funcions per processar-lo. Si el text està buit, es genera un error.
