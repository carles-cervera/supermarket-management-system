Detailed Description

La classe ScriptDialog és un SINGLETON i extén JDialog i representa una finestra modal que permet a l'usuari introduir un script per crear un inventari. Conté una àrea de text per escriure el script, botons per tancar la finestra i crear l'inventari, així com una interfície gràfica organitzada amb panells i separadors.
