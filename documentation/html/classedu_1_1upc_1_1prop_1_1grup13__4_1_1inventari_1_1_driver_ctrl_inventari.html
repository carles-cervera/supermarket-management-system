Detailed Description

Driver de CtrlInventari.

Aquest driver dona suport a dos modes d'interacció:

  • Mode interactiu: permet a l'usuari interactuar amb el sistema per mitjà d'opcions.
  • Mode scripting: permet a l'usuari interactuar amb el sistema per mijà d'instruccions ben definides.
